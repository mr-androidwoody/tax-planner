Two-Person Overflow Tests

Tax-aware mode · p1 shortfall spills to p2 · p1 empty → p2 covers all · proportional split

Tax-aware overflow logic: Each person draws SIPP up to their PA headroom first, then remaining shortfall is split proportionally (by remaining PA headroom, defaulting to 50/50 if equal). If p1's allocated share exceeds their available non-SIPP balance, the unmet amount spills over to p2 (and vice versa). If both persons' wrapper draws fall short, a final catch-all draws additional SIPP as a last resort. All tests use 2026-27 rules, growth=0, inflation=0. Tolerance £1.00.
